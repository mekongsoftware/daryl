Welcome to CRS
Complete Restaurant Solution

This software is a POC for restaurants and food trucks with these features:

  • Waiter/Waitress to take order using a tablet
  • Kitchen staffs can view and update order status in real time
  • Casher can view order status and print the final bill for customer
  • Customer can order online or from a tablet on the table (optional)
  • Multi-lingual support
  • Multi-restaurant support
  • Restaurant customization can be done quickly
  • Client can choose between cloud based or intranet deploymnent
